# Cub3D

Recreate the 1992 Wolfenstein 3D game.<br><br>
The project was done under MAC OS.<br>
Inputs are no longer avalable.<br>

## Usage

- Clone Repository
```bash
git clone https://github.com/vvaucoul/Cub3D && cd Cub3D
```

- Compilation
```bash
make
```

- Launch the game
```bash
./Cub3D maps/[Map].cub
```
